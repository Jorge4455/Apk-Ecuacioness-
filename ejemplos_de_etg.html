Ejemplos de ETG

U4: Ecuaciones de Tercer Grado. S3. Factorización de polinomios de tercer grado y resolución de ecuación de tercer grado. C3. Resolución de ecuaciones de tercer grado

Problema 1.

Resuelva la siguiente ecuación x(x-2) (x+1) =0

Solución 1:

Paso 1: Igualar a cero cada factor del lado izquierdo 

x=0;   x-2=0  x+1=0

Paso 2: Resolver para x moviendo la constante a la derecha añadiendo su opuesto en ambos lados 

x = 0                                    

x - 2 = 0

x - 2 + 2 = 0 + 2

x = 0 + 2

x = 2

x + 1 = 0

x + 1 - 1 = 0 - 1

x = 0 - 1

x = - 1

Respuesta de la ecuación 

x = 0; x = 2; x = - 1

 

Problema 2.

Resuelva la siguiente ecuación x3 +3x 2+2x = 0

Solución 2:

Paso 1: Sacar factor común de la expresión 

x3 +3x 2 +2x = 0

x3 +3x 2 +2x = 0

x (x 2 + 3x +2) = 0

Paso 2: Factorizar el resultado de la expresión 

x (x 2 + 3x +2) = 0

x (x + 1) (x + 2) = 0

Paso 3: Igualar a cero cada factor del lado izquierdo 

x = 0; x + 1 = 0; x + 2 = 0

Paso 4: Resolver para x moviendo la constante a la derecha añadiendo su opuesto en ambos lados

x = 0                                    

x + 1 = 0

x + 1 - 1 = 0 - 1

x = 0 - 1

x = - 1

x + 2 = 0

x + 2 - 2 = 0 - 2

x = 0 - 2

x = - 2

Respuesta de la ecuación

x = 0; x = - 1; x = - 2
